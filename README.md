# 📺 看电视 - 适老化视频播放器

> 专为老年人设计的安卓视频播放器，大按钮、大字体、极简操作

## 功能特点

- 🎯 **超大按钮** - 播放/暂停/前进/后退，手指粗也不怕按错
- 📝 **超大字体** - 剧名、集数一目了然
- 🔄 **30秒快进/后退** - 一步跳30秒，不用精准拖进度条
- 🖥️ **横屏全屏** - 自动横屏+沉浸模式，没有状态栏干扰
- 📂 **本地文件** - 不需要网络，视频放手机里就能看
- 🌙 **暗色护眼** - 深色背景，长时间看也不累

## 目录结构约定

在手机存储中创建如下目录，放入MP4文件即可：

```
/sdcard/看电视/
  ├── 乡村爱情故事/
  │   ├── 第01集.mp4
  │   ├── 第02集.mp4
  │   ├── 第03集.mp4
  │   ├── ... 
  │   └── cover.jpg        (可选：封面图)
  ├── 西游记/
  │   ├── 第01集.mp4
  │   ├── ...
  │   └── cover.png
  └── 还珠格格/
      ├── 01.mp4
      ├── 02.mp4
      └── ...
```

## 三个界面

### 1. 主页 - 选剧
大卡片展示所有电视剧，点击进入选集

### 2. 选集 - 选集数
大按钮展示每集，点击开始播放

### 3. 播放器 - 看视频
点击屏幕显示控制栏，3秒自动隐藏：
- ⏪ 后退30秒
- ⏯ 播放/暂停（最大按钮）
- ⏩ 前进30秒
- 进度条（可拖动）
- 🔙 返回选集

## 编译

```bash
cd 看电视
./gradlew assembleRelease
```

生成的APK在 `app/build/outputs/apk/release/`

## 安装

```bash
adb install app/build/outputs/apk/release/app-release.apk
```

## 支持的视频格式

MP4, MKV, AVI, MOV, 3GP, WebM（取决于安卓系统解码器）

## 最低系统要求

- Android 5.0 (API 21) 及以上
